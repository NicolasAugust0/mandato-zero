# 🎮 Mandato Zero: Crise Total

Jogo 2D de estratégia e gestão pública onde o jogador assume o controle de um governo por 4 anos, tomando decisões que afetam a economia, popularidade e estabilidade social.

---

## 🚀 Como jogar

1. Baixe ou clone o projeto  
2. Abra o arquivo `index.html` no navegador  
3. Tome decisões a cada turno  
4. Tente terminar com alta aprovação e caixa positivo  

---

## 🧠 Mecânicas principais

- Sistema de turnos (16 rodadas)
- Decisões com impacto direto
- Eventos aleatórios:
  - Enchentes
  - Secas
  - Greves
  - Ataques
- Sistema de tensão global (risco nuclear)
